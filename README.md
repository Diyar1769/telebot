# Валютный конвертер

Простой бот для конвертации актуальных валют. Пользователи выбирают валюты через инлайн кнопки и вводят сумму для конвертации.

## Доступные команды:
- `/start` – Запуск бота.
- `/help` – Список команд.
- `/list` – Список доступных валют.

## Как использовать:
1. Выберите исходную валюту.
2. Введите сумму для конвертации.
3. Выберите валюту для конвертации.
